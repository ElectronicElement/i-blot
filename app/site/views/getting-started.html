Blot is a blogging platform. It creates a folder in your Dropbox and publishes files you put inside automatically. There’s no publishing interface, just files and a folder. This means you can use your favorite text‐editor to write your blog.

Formatting your blog posts

Blot supports Markdown, including some of the features of its variants. Blot comes with a simple layout extension too. Blot also supports syntax highlighting for code. It knows how to render mathemtatics set in LaTeX.

Read the formatting guide 
# Formatting

You can use *Markdown*, a lightweight markup language for plain text files.

Use it for links, lists, headings, tables and more.

Drafts

Files in a folder called ‘Drafts’ are not published. Instead, Blot creates a preview which you can open in your browser.


Pages

Files in a folder called ‘Pages’ become pages instead of blog posts. They are added to your blog’s menu.

Metadata

Each blog post’s permalink and date is generated automatically from the file’s contents and location. You can override this metadata at the start of a file.

How to change a post’s date How to add tags to a post View the metadata reference 
Date: {{#date}}MMMM Do, YYYY{{/date}}
Permalink: /metadata

Metadata must start on the file’s first line and must be separated from the rest of the post by at least one blank line.

Make sure you leave a space after the colon.

Public files

All the files in your blog's folder are public and accessible to your readers. This makes Blot useful for serving static files.


Ignored files

Files in a folder called ‘Pages’ become pages instead of blog posts. They are added to your blog’s menu.
