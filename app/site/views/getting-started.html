Pages

Files in a folder called Pages are added to your blog’s menu instead of the list of blog posts. Use this feature to create an ’About’ or ’Contact’ page for your blog.

Drafts

Files in a folder called Drafts are not published. Instead, Blot creates a preview file which you can open in your browser to see how the post would appear on your blog.

Public files

The files in your blog’s folder are public and can be accessed by your readers. Use this feature to distribute files and embed media in your blog posts. Read more about serving static files.

Ignoring files

Blot skips files and folders whose name starts with an underscore. Ignored files do not become blog posts but they are public.

Metadata

The blog post's title, permalink and publish date is generated automatically from the file’s contents, name and location. You can override this metadata in the first few lines of a file:

Date: {{#date}}MMMM Do, YYYY{{/date}}
Permalink: /metadata

Metadata must start on the file’s first line and must be separated from the rest of the post by at least one blank line.

View all the metadata which you can specify at the start of a file. You can also create your own metadata and use them in your blog's template.

Dates

Blot uses the file’s creation date as its publish date by default. You can specify a different date in the file’s metadata like this:

Date: {{#date}}MMMM Do, YYYY{{/date}}

You can schedule a blog post for publication if you specify a date that's in the future. Blot will wait until then to publish it.

If you prefer to write your dates in a different order, please update your date preferences on your blog’s dashboard.

Tags

You can group posts which cover similar subjects using tags. You can tag a post in the metadata at the top of a file:

Tags: Getting started, Documentation

Tags are case-insensitive and may contain whitespace. Blot uses the most recent case when displaying a tag on your blog.

You can use brackets in a file or folder’s name to add additional tags to a blog post. For example, Ada.txt will have the tag 'Books' and Thief.txt will have the tag 'Films':


[Books]
  Ada.txt
[Films]
  Thief.txt

Permalinks

Blot generates a permalink, or permanent URL, for each post. By default, Blot uses a URL-friendly version of the post’s title. You can change the format of your blog’s permalinks on the dashboard or you can specify one in the file’s metadata:

Permalink: /how-to/change-a-url

If you change a post’s permalink, Blot will redirect requests to its former URL automatically.
