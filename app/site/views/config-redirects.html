Redirects

You can set up redirects for missing or moved pages on your blog. Your blog’s 404 log will help you discover URLs which need redirects. You can use RegEx to match multiple routes. For example, this rule redirects URLs starting with /post to /blog:

\/post\/(.*)  →  /blog/$1

Redirects are evaluated last in the order of Blot's routing. So if there is a blog post or file which matches the URL of a redirect, the redirect will not be evaluated.
