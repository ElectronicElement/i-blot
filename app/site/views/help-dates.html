Dates

A blog post’s publish date is the same as the file’s creation date by default. This usually is when you put the file inside your blog's folder.

Changing a post's date inside the file

The publish date is one of the properties of a blog post you can modify in the metadata at the start of a text-based file. Here’s an example blog post with a modified publish date:


Date: {{#date}}MMMM Do, YYYY{{/date}}

Blot supports a variety of date formats. The following formats are equivalent:

{{#date}}MM/DD/YYYY{{/date}}   {{#date}}MMMM D, YYYY{{/date}}
{{#date}}M.D.YYYY{{/date}}     {{#date}}YYYY-MM-DD 00-00{{/date}}

Changing a post's date using your folder

You can specify a post’s publish date in the name or path of the file. For example, all of these posts share the same publish date:

 {{#date}}YYYY{{/date}} /    {{#date}}MM{{/date}} /    {{#date}}DD{{/date}} /    Post.txt
 {{#date}}YYYY{{/date}} /    {{#date}}MM-DD-[Post.txt]{{/date}}
 {{#date}}YYYY-MM-DD-00-00[-Post.txt]{{/date}}

You can use underscores, dashes and dots to seperate the components of the date. Please note that a date specified in the file’s metadata takes precedence over one specified by its path.

Scheduling posts for publication

Blot will schedule a blog post for future publication if you specify a date in the future. You can adjust your blog’s time zone and other date‐related settings on the dashboard.
