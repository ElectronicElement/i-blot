Post metadata

Below is the list of blog post metadata which you can override at the start of a file. You can also define your own metadata. Blot uses this metadata to determine the properties of each post.



Date Defaults to the file’s creation date. Specify a publish date using one of the supported formats or by including the date in the file’s name.
Title Defaults to the text of the first heading in the blog post, or the file’s name.
Permalink Defaults to a URL-friendly version of the post’s title. You can change the general format of your blog’s permalinks, or specify one individually.
Thumbnail The source image for the thumbnails generated by Blot. Defaults to the largest image in the blog post. You can specify a URL, or a path to an image inside your blog’s folder.
Summary Defaults to the first sentence of the first paragraph in the file.
Tags Empty by default. Specify a list of tags, or use the file’s path.
Draft Defaults to ’No’. Specifying ’Yes’ will turn a blog post into a draft.
Page Defaults to ’No’. Specifying ’Yes’ will turn a blog post into a page.
Menu Defaults to ’Yes’ for pages. ’No’ hides a page from your blog’s menu.
