Permalink formats

Each post is blog assigned a permalink, or permanent URL. You can edit the format of these permalinks on the dashboard. The permalink's format can include some of the blog post's properties. Here are a few examples:

{{=[[ ]]=}}
The permalink format... produces URLs like...
{{slug}} www.example.com/title-of-the-post
blog/{{slug}} www.example.com/blog/title-of-the-post
{{YYYY}}/{{slug}} www.example.com/2017/title-of-the-post
{{YYYY}}/{{MM}}/{{D}}/{{slug}} www.example.com/2017/10/3/title-of-the-post
{{slug}}-{{MM}}-{{YYYY}} www.example.com/title-of-the-post-10-2017

Here is the full list of blog post properties that you can use, in addition to the date tokens for the post's publish date:

Property Value
{{slug}} A URL-friendly version of the post's title
{{slug-without-diacritics}} {{slug}} without diacritical marks
{{name}} The file's name, lowercased
{{name-without-extension}} {{name}} without the file extension
{{path}} The file's path, lowercased
{{metadata}} An object containing metadata specified in the file
{{size}} The file's size in bytes
{{dateStamp}} A unix datestamp for the post's publish date
{{created}} A unix datestamp for the file's creation date
{{updated}} A unix datestamp for the file's modified date
{{more}} True if the post's teaser is shorter than the full post
{{page}} True if the file is a page
{{menu}} True if the file is a page and is on your blog's menu
