Creating a page

The easiest way to create a page is to make a new folder called Pages and put a file inside which contains the page’s content:

# About

If this file is in a folder called ’Pages’, Blot will add it to your blog’s menu instead of turning into a post. You’ll have an about page!

Using metadata

You can also use a post’s metadata, instead of a folder:

Page: yes
Menu: no

Since this file’s metadata contains ’Page: yes’, Blot will turn this file into a page even if it isn’t inside a folder called ’Pages’. And since this file’s metadata contains ’Menu: no’, it will not appear on your blog’s menu.
