Post properties

Below is the list of properties which Blot generates for every blog post. These properties are determined by the file from which the blog post is created.

You can use any of these properties in your template as tags. Note that some of the variables, like html require three handlebars rather than two because of how Mustache handles escaping.

Property Description
html The full HTML of the blog post, generated by Blot from the file in your blog's folder
path The file's absolute path with the blog's folder as root, preserving the case where possible. For example:
"/Posts/2018/Post.txt"
title The post's title, generated from the title property in the file's metadata, from a title tag in the file, or from the file's name. For example:
"Your post's title"
url Path section of the URL to the post. For example:
"/your-blog-post"
name The file's name, preserving the case where possible. For example:
"Post.txt"
titleTag HTML of the tag which was used to generate the post's title, if it exists in the file. For example:
"<h1>Your post's title</h1>"
body HTML of the entry, excluding the titleTag
summary Text of the first paragraph of the entry.
teaser HTML of the title and the first few items in the entry. You can specify the end of the teaser by inserting the special tag more into your blog post's file. Everything before more will be included in the teaser.
teaserBody teaser without the title.
more Boolean indicating whether teaser differs from html. This is useful for determining whether or not to show 'read more' links.
slug The value of the metadata property permalink. Same as URL.
permalink The value of the metadata property permalink. Same as URL.
size The size of the file.
date Human readable publish date for the entry
tags List of tags for the entry
menu "true" if the entry is a page on the menu, "false" if not
page "true" if the entry is a page, "false" if not
deleted "true" if the entry is deleted, "false" if not
draft "true" if the entry is a draft, "false" if not
scheduled "true" if the entry is scheduled for future publication, "false" if not
thumbnail object containing the thumbnails for the entry, if they exist.
dateStamp Timestamp
created Timestamp
updated Timestamp
metadata object containing the metadata for the entry, if they exist.
id Alias for the path property, which uniquely identifies the post.
guid Alpha numeric identifier
