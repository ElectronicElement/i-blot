How to embed an image in a blog post

There are many ways to embed an image in a blog post on Blot. The best way in your particular case will depend on a few things: - Is the image already on the internet? - What type of file is your blog post?


Using a Word Document
Just drag and drop the image into the Word Document.


Using Markdown to embed an image on the internet
Here's the syntax to embed an image using Markdown. Please note that the image on the internet must be public.
