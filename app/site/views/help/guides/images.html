How to insert an image into a post

There are many ways to embed an image in a blog post on Blot. The best way in your particular case will depend on the type of file that you use to create the blog post.

Using a Word Document

Just drag and drop the image into the Word Document.

Using a text file and Markdown

Here's the syntax to embed an image that's already on the internet. Please note the exclamation point!

![Image's title](http://url.of/the/image.jpg)

What if you want to use an image that's in your site's folder? First, you will need to make sure that your image's filename begins with an underscore, or that it's inside a folder whose name begins with an underscore. This is neccessary because Blot turns images inside your blog's folder into blog posts. So let's say your blog's folder looks like this:

Your blog Name _Images First.jpg Posts Your post.txt _Second.jpg      

Notice that First.jpg is inside _Images, whose name begins with an underscore and _Second.jpg's name begins with an underscore. This ensures that Blot will not turn them into blog posts in their own right. You can then embed the various images in Your post.txt like this:

![First](../_images/_.jpg)
![Second](/_images/second.jpg)

![Apple](/_images/apple.jpg)
![Peach](_peach.jpg)


Notice how paths are case-insensitive. Paths are resolved relative to the source file.
