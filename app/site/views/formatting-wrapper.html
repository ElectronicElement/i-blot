Formatting guide

Blot offers a variety of standard tools to add formatting to your blog posts. You can use Markdown to add links, headings, lists and much more. Blot also renders mathematics set in \(\TeX\) and provides syntax highlighting for code snippets.

The formatting described below is available for blog posts created from  Text files,  Markdown and  Word Documents.
