Formatting guide

You can use a variety of standard tools to add formatting to your blog posts. Text files can include Markdown to add links, headings, lists and much more. You can mark up mathematics in \(\TeX\). Note that Blot can turn HTML files into blog posts.

  • Markdown A lightweight markup language with plain text formatting syntax.
  • \(\TeX\)A math typesetting library.
  • LayoutA custom Markdown extension which offers simple layout directives.
