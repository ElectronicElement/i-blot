Clients

Your blog's folder is synchronized with Blot by a client. These clients are available:

  • Dropbox
  • Git

You can switch seamlessly between clients at any point.

Please contact me if there's a client you'd like me to add. Blot can theoretically support any system which is able to synchronize a folder over the internet. I'll be working on supporting FTP next, then Google Drive and OneDrive.
