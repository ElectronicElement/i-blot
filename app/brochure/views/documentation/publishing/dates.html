Dates

A blog post’s publish date is the same as the file’s creation date by default. You can specify a different date in the file’s metadata or in its path. The publish date is one of the properties of a blog post you can modify in the metadata at the start of a file. Here’s an example blog post with a modified publish date:

Date: {{#date}}MMMM Do, YYYY{{/date}}

Blot supports a variety of date formats. The following all produce a blog post with the same date:

{{#date}}MM/DD/YYYY{{/date}}   {{#date}}MMMM D, YYYY{{/date}}
{{#date}}M.D.YYYY{{/date}}     {{#date}}YYYY-MM-DD 00-00{{/date}}

Using the file’s path

You can modify a post’s publish date using the file’s name or path. Please note that a date specified in the file’s metadata takes precedence over one specified by its path. For example, all four of these posts share the same publish date:

 {{#date}}YYYY{{/date}} /    {{#date}}MM{{/date}} /    {{#date}}DD{{/date}} /    Post.txt
 {{#date}}YYYY{{/date}} /    {{#date}}MM{{/date}} /    {{#date}}DD-[Post.txt]{{/date}}
 {{#date}}YYYY{{/date}} /    {{#date}}MM-DD-[Post.txt]{{/date}}
 {{#date}}YYYY-MM-DD-00-00[-Post.txt]{{/date}}

Schedule a blog post for future publication. If you specify a date in the future, your post will be published then.

You can use underscores, dashes and dots to seperate the components of the date.
