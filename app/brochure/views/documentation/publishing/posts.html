Blog posts

All of these files become blog posts automatically when you put them somewhere inside your blog’s folder. Any changes you make are applied automatically to your site. See Drafts

 Text and Markdown .txt  .md Use the full range of markdown features in these files, including footnotes.
 Word Documents .docx  .odt Most of the typographic formatting is stripped.
 Bookmarks .webloc  .url Drag URLs from your browser's address bar to create these files. Blot turns them into a simple link.
 Images .png  .jpeg  .gif
 HTML .html   .htm

Add formatting to your posts

. Blot supports a variety of standard tools to add formatting to your blog posts. You can use Markdown to add links, headings, lists and much more. Blot also renders mathematics set in \(\TeX\) and provides syntax highlighting for code snippets. Read the formatting guide 

# Formatting

You can use *Markdown*, a lightweight markup language for plain text files.

Use it for links, headings, lists and much more.

Prevent a file becoming a blog post.

Files and folders whose name starts with an underscore are ignored. Ignored files don’t become blog posts themselves but they can be embedded in other posts.
