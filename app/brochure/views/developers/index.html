Blot's template system offers you, the developer, control of every line of HTML, CSS and JavaScript delivered to your readers.

Here you’ll find a guide to developing templates for Blot. You will need a working knowledge of HTML and CSS. Blot's templates will be immediately familiar to those who have developed for other platforms.

If you’d like to hire a developer or designer to create a template for you, contact me. I'll put you in touch with professionals who are familiar with Blot's templating system.

Getting started

You can create a new template on the your blog’s dashboard. You can make changes to a template using the online editor, or you can sync the template’s source files to your blog’s folder and edit them locally.

To familiarize yourself with Blot's templates I recommend creating a copy of a template on the dashboard and looking through its source code. All of Blot’s templates are dedicated to the public domain so you can modify and share them freely.

How Blot works

Behind all the fan-fare, Blot is a static file server. It serves the contents of your folder to your readers. Above the static file server is Blot's template system, which lets you do cool things with the list of entries Blot generates from your folder

Read more about how Blot works

Rendering templates

Blot’s templates will be familiar to those who have worked with other blogging platforms. They are written in Mustache and consist of tags which look like \{{this}}. These tags are replaced with the data generated by Blot when the template is rendered.
