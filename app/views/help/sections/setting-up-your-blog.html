Migrating to Blot from other platforms

I've written scripts to import blog posts from Tumblr, Wordpress, Jekyll and Squarespace. These scripts are not yet connected to a web interface but if you contact me, I can run them on your blog. The scripts generate a folder of Markdown files.

Using a personal domain

Once you own a domain, add it to your account on the settings page.

If you’d like to use a subdomain e.g blog.example.com or www.example.com create a CNAME record for that subdomain which points to blot.im.

If you’d like to use an apex domain e.g. example.com then create an ALIAS record which points to blot.im. If you cannot create an ALIAS record, then use a CNAME record with the www. subdomain and redirect the apex domain.

Redirects

You can set up URL redirects on the redirects page. This is useful for migrating an existing blog to Blot. The 404 log page will help you work out what needs to be redirected. You can use Regular Expressions to match multiple routes.
