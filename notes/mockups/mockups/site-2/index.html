Blot is a blogging platform.
Know how to use Microsoft Word? Then you can use Blot to run a blog. But you don’t have to use Microsoft Word ... Blot lets you blog with your favorite text-editor.

Here’s how it works: Blot creates a folder in your Dropbox and publishes files you put inside. Text files, images, Markdown, bookmarks, Word Documents and more become blog posts. Publishing could not be lazier.
