Know how to use Microsoft Word? You can use Blot to run a blog. Blot creates a folder in your Dropbox and publishes files you put inside automatically. Images, text files, Markdown, bookmarks and more become blog posts.

Members pay $30 per site, per year. This fee keeps Blot fast and reliable. Blot comes with 8 templates. They're designed to be easy to customize. Some work well with lots of photography, others are better suited to text. If you're comfortable with HTML and CSS, you can also create your own from scratch.
