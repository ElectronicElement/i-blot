Quick start8

Move the file containing your post into Blot’s folder in your Dropbox. Blot will publish it to your blog. Editing the file will update the post!

What files can Blot publish?8

Blot can turn text files (.txt), markdown files (.md) and HTML files (.html) into blog posts. Blot will also convert image files (.jpg, .gif and .png) into image posts automatically.

Can I create folders inside Blot's folder?8

Yes, Blot doesn’t mind how you organize files inside its folder. I have created subfolders for each year to keep things tidy!

Pages8

Adding a page is similar to publishing a post — just create a text file containing the page’s content then add this line at the top of the file: 

Page: yes

Blot will add a link to this page to your blog’s menu. This is useful for creating about and contact pages.

Changing a post’s date8

You can specify a post’s date the top of its file. Blot supports plenty of formats. 23/06/2012, 23.6.12 and 23-6-2012 all work well:

Date: June 23rd 2012

Drafts and Previews8

If you create a folder called ‘Drafts’ inside Blot’s folder, Blot will not publish files you put inside. Instead, Blot will create a file containing a preview of that post. If you open the preview in your web browser, you will see exactly how that post would appear on your blog.

To publish the post, just move the file to another directory in your blog folder.

Public files8

If you create a folder called ‘public’ (or ‘Public’), Blot will make files you put inside publicly accessible and will not turn them into blog posts.

For instance, if your blog was example.com, files in your public folder would be accessible at example.com/public. This feature is useful for offering downloads, or embedding images inside posts.

Scheduled posts8

If you specify a date at the top of the file, Blot will wait until then to publish it. If you want to cancel the post, just delete the file.

Date: March, 13th 2020

Permalinks8

You can specify a post’s permalink at the top of its file. For example, to create a blog post with the URL example.com/yosemite, add this:

Permalink: yosemite

Tags8

You can specify tags at the top of a post’s file. The tags are case insensitive and can contain multiple words:

Tags: literature, Stendhal, book reviews

Restoring a deleted entry8

If you put a post’s file back in Blot’s folder, Blot will restore the post to your blog, including its former URL. If you can’t find the file, try using Dropbox to recover it.

What happens if my blog post is too large?8

Blot ignores text files larger than 300kb (~75,000 words) in size. If one of your entries is larger than this, please consider splitting it into multiple entries for the sake of your readers. Please contact me if you need to lift this limit.
